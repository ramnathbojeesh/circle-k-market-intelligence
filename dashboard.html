Methodology, scoring & auditability

Every metric on this dashboard defines what it measures, its sources, and whether it is platform-reported, calculated, modelled or simulated. Headline numbers reconcile to the tables shown; theme totals are non-mutually exclusive where noted.

Scoring framework — five transparent scores (0–100), editable weights

A · Social Momentum MODELLED

Sum must equal 100%. Live inputs: mention growth/acceleration from weekly pulls; creator growth & organic share from document sampling (directional); search growth not connected this build (weight redistributed, noted).

Methodology fusion (jedis-methodology)

TPV (Black Swan logic): 0.30×size + 0.40×growth + 0.30×forecast, normalised within category — trajectory beats size. Velocity/relevance (Tastewise logic): wave vs fad via multi-period persistence. Lifecycle + drivers (Mintel): dirty soda → Experiences; Poppi/Olipop → Wellbeing; Ghost 7UP → Identity/nostalgia. Megatrend fit (Euromonitor): Convenience + Healthy Living — durability check. Tribe opportunity (Soprism index): Pen + 3(Pen·Aff) + 3·Cov + 3(Cov·Aff) + Aff, normalised. Stickiness = 0.6×tribe-fit + 0.4×momentum. All reconstructed from public vendor descriptions — analytical inspiration, not proprietary data.

Entity resolution

Product mentions map to canonical products via boolean variants (e.g. "Ghost" / "Ghost Energy" / "Ghost Sour Patch" → Ghost Energy family). Match confidence is surfaced per product in the Data conf. column; low-confidence matches are not merged (e.g. Ghost Jan-2026 excluded when the boolean broadened to bare "Ghost").

Metric definitions

MentionsCalculated — documents matching the generated boolean, US country filter, per Meltwater unified retrieval. Weekly buckets = week beginning Monday.
Engagement / reach / viewsPlatform-reported on individual posts; definitions differ per platform — never summed across platforms here. Where a normalised index is needed, it is labelled and the method stated.
SentimentMeltwater document-level classification; splits shown as % of sentiment-classified volume; "unknown" excluded and disclosed.
Unique creatorsDirectional — from document sampling, not census counts. Flagged wherever used.
GeographyUS country filter = US-level proxy. State labels applied only when the source itself is state-anchored (e.g. TX AG release) = State-confirmed. No state-inferred claims made this build.
Trend stageCalculated — multi-criteria rules on Tab 5; never volume alone.
Proponent classCalculated from account ownership, disclosure labels, tier, posting pattern; sponsorship labels are evidence-gated.
Audience groupsInferred from bios/hashtags/content patterns — modelled, not verified.
DuplicatesSyndicated news counted per Meltwater dedup defaults; broadcast transcripts inflate neutral share (disclosed below).
